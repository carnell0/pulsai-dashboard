# PulsAI Dashboard

Ce projet est un Dashboard CRM moderne construit avec Next.js, Tailwind CSS et Framer Motion.

## Fonctionnalités

*   **Architecture Next.js App Router**
*   **Design System** : Couleurs personnalisées, polices Ubuntu, ombres douces.
*   **Sidebar Hybride** : Navigation rétractable avec animations fluides.
*   **Pages** : Dashboard, Conversations, Tickets, Contacts, Campagnes, Analytique, Paramètres.
*   **Données** : Mock data riche pour simuler un environnement de production.

## Démarrage

```bash
npm run dev
# ou
yarn dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) pour voir le résultat.
